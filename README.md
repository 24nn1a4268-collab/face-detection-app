# Face Detection App
A simple computer vision project built using Python and OpenCV to detect human faces from webcam or images using Haar Cascade Classifier.

## Features

* Detect faces in real time using webcam
* Detect faces from images
* Draw rectangles around detected faces
* Beginner-friendly OpenCV project

## Technologies Used

* Python
* OpenCV
* Haar Cascade Classifier

## Run the Project

```bash
python face_detection.py
```
